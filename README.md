# sublime-text

These files are from my sublime text config.

## Setup

For Sublime Text 4

1. Open Sublime Text.

2. Open the Command Palette:
    Windows/Linux: Ctrl + Shift + P
    macOS: Cmd + Shift + P

3. Type "Install Package Control" and select the option to install it.

4. Restart Sublime Text.
